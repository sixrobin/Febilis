- Id
- Category (weapon, usable, quest, misc, etc.)


Classe Inventory
- Liste de InventoryItem
	- est un Item
	- a une quantité
	- paramètre "droppable" pour les objets de quête/stuff de base du perso (épée, potions)
- Méthode Add()
	- vérifie qu'un objet du même Id n'est pas déjà dans l'inventaire, et si c'est le cas, alors augmenter la quantité de celui-ci
- Méthode Drop()
- Méthode Remove() (pour supprimer de l'inventaire sans drop par terre).
